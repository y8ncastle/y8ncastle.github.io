ICDL Professional Computing Python 자격 취득



지난 2019년 11월 4일부터 13일까지 학교에서 ICDL Computing Python 교육을 받았다.

3학년 때 처음으로 Module 3, 4, 6, 7 자격증을 취득했는데 이번에도 학교에서 동일한 방식으로 교육을 제공해줘서 들을 수 있었다.

아무래도 국제 컴퓨터 자격증인만큼 가격이 가격인지라 무료로 기회를 제공해준다고 했을 때 덥석 무는게 이득이라 생각해서 참여하게 되었고, 요즘 파이썬이 대세라고 하는 만큼 관련 자격증도 취득해놓으면 나중에 취업할 때도 도움이 되지 않을까 싶었다.


사실 한국에서는 어느 기업에서 가산점을 주는지는 모르겠지만 국제 자격증이라는 점에 혹해서… 하하


어쨌든 Computing Python은 코드보다는 컴퓨터 개론에 집중된 느낌이었고 난이도도 다른 컴퓨터 자격증들에 비하면 엄청 쉬운 편에 속했다.





약 2주 정도 교육을 받고 2019년 11월 18일에 학교 고사장에서 공식적으로 시험에 응시했는데 360점 만점에 350점을 받아 합격했다!

시간은 45분 중에 대략 15~20분 정도 쓴 것 같고 합격 컷은 275점 이상이었다.

모의고사에서 풀었던 유형의 70% 이상은 거의 동일하게 나왔다고 말할 수 있을 정도로 유형 파악이 중요했던 것 같다.





이제 기존 Module 3, 4, 6, 7까지 더해서 ICDL 자격증은 총 5개가 되었고, Professional 자격증과 모든 ICDL을 통합한 Profile Certificate를 추가로 신청하려고 한다.

파이썬 과목도 수강하면서 동시에 자격증도 취득하다 보니 파이썬에 대해 더 많은 관심이 생기게 되었고, 인공지능이나 토큰 제작에도 파이썬 언어가 자주 쓰인다고 하니 C언어보다도 더 많은 시간을 투자해서 공부해야겠다.
