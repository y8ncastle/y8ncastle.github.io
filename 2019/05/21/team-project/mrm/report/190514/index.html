주간 진행 보고서 (5월 3주차)

2019.05.14 (화)

이번 주는 원래 앱 내장 데이터베이스 설계가 계획되어 있었지만 27일에 있을 3차 발표 때문에 PPT를 제작하려고 한다.

금요일에 교수님께 1차 검사를 받고 다음 주 중으로 한 번 더 수정 작업을 거칠 예정이다.

오늘은 출근 후 I2C 통신 프로토콜 PPT 내용 전체 수정을 마무리하였다.

차후 장치 부분에 대한 내용을 추가로 수정해야 할 가능성도 있지만 일단 기존에 수정해야 할 내용들은 모두 수정했다. 해당 자료 마무리 이후부터는 계속해서 PPT를 구상하고 제작하고 있다.

3차 계획 PPT를 구상하는데 있어 1, 2차보다 더 쓸 내용이 없는 것 같다.

물론 한 달 동안 아무것도 하지 않은 것은 아니지만 일단 제품을 구입해야 무언가를 할 수 있을텐데 제품을 아직도 구입하지 못하고 있다보니 한 달 동안 한건 아두이노로 휠과 센서 테스트를 해봤다는 것, 통신 프로토콜에 관한 내용을 조사했다는 것, 앱 데이터 관리 화면을 디자인했다는 것 정도인 것 같다.

세부적인 내용없이 포괄적으로 진행하다보니 PPT에 들어갈 내용이 그렇게 많지 않아서 일단 만든 후에 교수님과 상담을 진행해보고 필요한 내용은 추가할 수 있도록 해야겠다.



2019.05.15 (수)

어제에 이어 캡스톤디자인 3차 발표 PPT를 제작했다.

추가적으로 I2C 통신프로토콜 관련 PPT를 재수정했다.

통신 프로토콜 조사와 정리로 거의 한 달을 쓴 것 같은데 계속해서 수정해야 할 내용들이 생겨서 생길 때마다 수정을 진행하고 있다.

아마 장치 부분을 포함해서 2~3번 정도는 더 수정해야 완성이 될 것 같은 느낌이다.

아직 수정해야 할 내용이 조금 남아있기는 하지만 금요일에 교수님을 먼저 찾아봬야 하기 때문에 다음 주까지 마칠 예정이다.

I2C 통신프로토콜 PPT 수정 작업 이외에 3차 발표 PPT 제작 부분에서는 기본적인 틀은 완성해두었다.

다만 스크립트와 내용을 동시에 쓰려고 하다보니 조금 버벅거리는 경우가 생겨서 일단 발표할 그림 형태만 대충 만들어두고 스크립트는 이번 주 주말이나 다음 주 중으로 완성할 예정이다.

틀은 짜놨지만 아무래도 하드웨어 쪽이든 소프트웨어 쪽이든 이번엔 통신 이론 조사 이외에는 진행한 내용이 뚜렷하지 않다 보니 PPT를 완성하는 데 조금 어려움이 있을 것 같다.



2019.05.16 (목)

오늘은 어제 구상한 틀을 토대로 캡스톤디자인 3차 PPT를 작성했다.

팀에서 소프트웨어 관련 연구를 위주로 하다보니 하드웨어에 대한 내용을 어떻게 구상해야 할지 머리가 복잡하고 생각도 잘 나질 않았는데 하드웨어를 맡고 있는 같은 팀원 친구가 이전에 많은 것들을 준비해두어서 정리하는데 상당히 편했다.

내일 교수님을 찾아뵙기로 예정되어 있었는데 교수님께서 메일로만 우선 전송하라고 하셨기 때문에 특별히 다른 말씀이 없으시면 내일이나 다음 주 중으로 마무리 짓고 바로 3차 발표를 진행할 예정이다.

오늘 PPT를 구성하는 부분에서는 대부분의 내용을 하드웨어 파트로 구성했다.

원래 1, 2차 발표 때에도 하드웨어를 위주로 작성해야 하는 것이 맞기는 했지만 안드로이드 앱 개발 이외에는 마땅히 할 수 있는 것들이 없었기 때문에 관련 내용들을 넣을 수가 없었다.

이번 3차 발표가 끝나는대로 부품을 받을 예정이고 지금은 기존에 회사 내에 있는 부품들을 이용해서 미리 연결해보고 테스트해보는 작업 등을 주로 작성했다.

앞으로 제품 조립 후에 안드로이드 앱 연동 실험이 성공적으로 끝날 수 있도록 많은 노력을 해야 할 것 같다.



2019.05.17 (금)

오늘 캡스톤디자인 3차 발표 PPT를 최종적으로 완성했다.

교수님께도 이메일로 제출드리고 특별한 일이 없으면 수정하지 않아도 될 것 같다.

기존 2차 발표 전까지는 약간 뜬 구름 잡는 것 같은 애매한 느낌을 가지고 발표 준비를 했었는데 이번 3차 PPT는 훨씬 더 많은 내용이 들어가고 확실한 실험 결과들만 담을 수 있어서 더 나은 것 같다.

PPT 작성 마무리 이후에 I2C 프로토콜 PPT도 다시 수정했는데 고쳐야 할 부분에 대한 내용이 모호해서 일단 내가 고칠 수 있는 선에서 해결하였다.

차후 추가 수정이 필요할 지도 모르겠다.

3차 PPT 관련해서는 어제 다 적지 못했던 소프트웨어 부분을 완성하고 전체 스크립트까지 짰다.

소프트웨어 부분은 세부적인 내용이 있기는 했지만 요약해서 보여주는 부분이 조금 부족한 것 같아서 그 부분을 조금 더 보완하였고 PPT 대부분의 내용이 하드웨어로 이루어져 있다보니 하드웨어 파트 스크립트는 팀원들과 함께 토의하면서 작성할 수 있었다.

앞으로 제품을 받는대로 바로 제작하고 앱 연동 방법도 슬슬 찾아봐야 할 것 같다.
