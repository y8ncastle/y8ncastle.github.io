#소수찾기

[백준] 1978 - 소수 찾기

5월 10 2019 Baekjoon Online Judge 2분 소요 (약 332 단어)



문제 정보

  • 문제 이름: 소수 찾기
  • 문제 번호: 1978
  • 문제 유형: 알고리즘 - 수학 / 에라토스테네스의 체
  • 풀이 언어: C
  • 제출 시간: 2019년 5월 10일 21시 36분 26초
  • 원본 링크: https://www.acmicpc.net/problem/1978

문제

주어진 수 N개 중에서 소수가 몇 개인지 찾아서 출력하는 프로그램을 작성하시오.

입력

첫 줄에 수의 개수 N이 주어진다. N은 100이하이다. 다음으로 N개의 수가 주어지는데 수는 1,000 이하의 자연수이다.

출력

주어진 수들 중 소수의 개수를 출력한다.

결과 - 소스코드 및 랭크





메모

이 문제도 기존에 틀렸던 문제였는데 다시 한번 생각하면서 문제를 풀었다.

순위가 저 정도로 떨어지는 것을 보니 좋은 코드는 아니지만 그래도 기존에 6번이나 틀렸던 것에 비하면 통과했다는 것만으로도 만족스럽다. 마지막 한 번은 실수로 테스트 코드를 그대로 집어넣었다가 틀렸지만 바로 수정해서 확인 처리되었다.

소수 구하는 문제가 생각보다 많던데 조금 더 빠르게 계산할 수 있는 코드를 생각해봐야겠다.
