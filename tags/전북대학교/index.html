#전북대학교

[백준] 14624 - 전북대학교

5월 18 2019 Baekjoon Online Judge 3분 소요 (약 414 단어)



문제 정보

  • 문제 이름: 전북대학교
  • 문제 번호: 14624
  • 문제 유형: -
  • 풀이 언어: C
  • 제출 시간: 2019년 5월 18일 17시 24분 59초
  • 원본 링크: https://www.acmicpc.net/problem/14624



문제

전북대학교의 심볼은 균형과 조화, 지성과 이상을 향한 방향성과 목표를 나타낸다. 절제된 한국적 아름다움을 꾸밈없는 소박함과 여백을 통해 시각화하였으며, 심볼의 방향에 따라 한국적인 대학, 학문에 정진하는 대학, 미래로 나아가는 대학의 의미를 포함하여 성장을 넘어 성숙의 대학으로 나아가는 전북대학교의 철학과 비전을 상징한다.



입력

정수 N (3 ≤ N ≤ 50)을 입력받는다.



출력

입력 N이 홀수인 경우 ‘*’을 이용해 가로의 길이가 N인 전북대학교 심볼을 출력한다. (예제 참고)

짝수인 경우, I LOVE CBNU를 출력한다.



결과 - 소스코드 및 랭크




메모

문제를 풀고 아무리 봐도 출력이 잘 나왔는데 출력 형식이 틀리다는 말만 계속 반복되어 포기하려다가 출력 형식이 뭐가 잘못된 것인지 검색을 해봤는데 각 줄의 마지막 별 뒤에는 공백이 나오면 안 되는 것이었다.

처음에는 공백을 각 줄 마지막 칸까지 출력하도록 했는데 그런 규칙이 있다는 것을 알게 된 후로 break 문을 이용해서 중간에 출력을 중단하도록 했다.
