[백준] 14490 - 백대열

5월 13 2019 Baekjoon Online Judge 2분 소요 (약 346 단어)



문제 정보

  • 문제 이름: 백대열
  • 문제 번호: 14490
  • 문제 유형: -
  • 풀이 언어: Java
  • 제출 시간: 2019년 5월 13일 22시 7분 46초
  • 원본 링크: https://www.acmicpc.net/problem/14490

문제

대열이는 욱제의 친구다.

  • “야 백대열을 약분하면 뭔지 알아?”
  • “??”
  • “십대일이야~ 하하!”

n:m이 주어진다. 욱제를 도와주자. (…)

입력

n과 m이 :을 사이에 두고 주어진다. (1 <= n, m <= 100,000,000)

출력

두 수를 최대한으로 약분하여 출력한다.

결과 - 소스코드 및 랭크





메모

오늘은 종일 멘붕이었다..

오후 5시쯤부터 스택 관련 문제를 코딩하기 시작했는데, 자료구조를 배우지 않고 알고리즘 강의를 듣는 바람에 스택을 제대로 사용해보는건 처음이기도 했고, 3시간 반 동안 해결을 못하다가 결국 다른 문제로 넘어갔다.

기본 입출력 케이스에 나오는 건 다 맞췄는데 안 되는 이유가.. ㅠㅠ

이 문제 전에도 한 문제 실패하고, 이 문제를 보면서 아.. 최대공약수로 풀면 되겠구나하고 간단히 생각하고 풀었는데 생각보다 시간이 너무 길게 나왔다. ㅠㅠ 코딩 실력 최악이다.. 진짜 공부 열심히 좀 해야겠다.
