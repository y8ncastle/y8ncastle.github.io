온북스 (On Books) 앱 프로토타입 개발



본 게시글은 개발 현황을 공유하기 위해 작성되었습니다.



현재 개발 완료 사항

2019.06.05

  • 메인화면 프래그먼트 분할 (상단 메뉴 카테고리 구분)

2019.06.06

  • 메인화면 레이아웃 일부 배치
  • 메인화면 커스텀 리스트뷰 구성
  • 네이버 국어사전 및 지식백과 웹뷰 연동

2019.06.08

  • 메인 커스텀 리스트뷰 디자인
  • 책 정보 등록하기 다이얼로그 추가
  • 갤러리 이미지 로드 테스트

2019.06.18

  • 책 정보 등록하기 다이얼로그 입력 기능 적용
  • 프래그먼트 간 데이터 전달 적용
  • 커스텀 리스트뷰 프로그레스바 추가

2019.06.19

  • 리스트뷰 책 페이지 추가 기능
  • 페이지 추가시 커스텀 리스트뷰 프로그레스바 퍼센트 적용
  • 메인화면 데이터베이스 테이블 구현
  • 나의 독서 컬렉션 빈 화면 구상

2019.06.20

  • 독서 100% 달성시 트로피 마크 표시 및 추가 버튼 삭제
  • 트로피 클릭시 독서 컬렉션으로의 데이터 이동 기능
  • 온북스 개발 현황 트래커 작성

2019.07.01

  • 책 정보 등록시 이미지 추가 기능 구현
  • 내장 메모리 접근 퍼미션 적용

2019.07.02

  • 책 정보 등록시 이미지 추가 기능 보완
  • 메인 오류 수정, 독서 컬렉션 연동

2019.07.06

  • 메인 컬러 전체 수정 (6CBB5C → 499360)
  • 프로그레스바 55% 이상시 폰트 색상 하얀색으로 수정
  • 독서 타이머 레이아웃 구성

2019.08.07

  • 독서 타이머 기능 보완

2019.08.10

  • 독서 타이머 기능 추가 보완
  • 시간 작동 확인, 명언 추가, 일시정지 기능 구현

2019.08.12

  • 기록 테이블 조인 연동 (Data 테이블 추가)

2019.08.14

  • 기록 테이블 조인 연동 보완 (데이터 값 전달 테스트)
  • 여러가지 예외처리 추가

2019.09.08

  • 챗봇 기능 추가 (메인화면 플로팅)
  • 100% 달성시 폭죽 다이얼로그 기능 추가
  • 퍼센트에 따른 사람의 움직임 기능 구현

2019.09.14 (프로토타입 개발 종료)

  • 기기별 레이아웃 개선 1차 작업 진행 후 종료
  • 타이머 스레드 핸들러 오류 수정

2019.09.23 (추가 요청 사항 개선 후 플레이스토어 업로드 완료)

  • 메인 상단바 이름 및 위치 수정
  • 챗봇 뒤로가기 안내 토스트 추가
  • 온북스 앱 아이콘 추가
  • 책 평가 척도 (레이팅바) 표시 위치 수정



현재까지 개인 일정을 제외한 순수 개발일수는 총 16일 입니다.
