[모두의 딥 러닝] Machine Learning의 용어와 개념



머신러닝

  • Limitations of explicit programming(일반적인 소프트웨어)을 해결하고자 연구한 방법

  • 프로그램이지만 개발자가 일일이 어떤 현상에 대해 정하지 않고 프로그램 자체가 직접 데이터를 학습하는 능력을 갖게 하는 것



Supervised vs Unsupervised

  • Supervised: Learning with labeled examples (이미 있는 데이터를 제공하여 학습)

  • Unsupervised: Un-labeled data (미리 데이터를 정해주기 어려운 경우)



Types of Supervised Learning

  • Regression: 큰 범위 내에서 예측 (ex. Predicting final exam score based on time spent)

  • Binary Classification: 두 개 중에 하나를 고르는 것 (ex. Pass or non-pass based on time spent)

  • Multi-label Classification: 여러 가지 정해진 값 중에서 하나를 고르는 것 (ex. Letter grade based on time spent)
